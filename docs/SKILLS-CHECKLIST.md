## 🔧 安装脚本

### 快速安装 (全部)
```bash
# 保存为 install-all.sh
#!/bin/bash
SKILLS="gog obsidian nano-pdf openai-whisper mcporter trello clawhub tmux"

for skill in $SKILLS; do
    echo "Installing $skill..."
    npx clawhub install $skill
    echo "Waiting 30 seconds..."
    sleep 30
done

echo "All skills installed!"
```

### 按角色安装
```bash
# PM 角色
npx clawhub install gog obsidian nano-pdf openai-whisper trello

# PO 角色
npx clawhub install gog obsidian nano-pdf openai-whisper

# ARCH 角色
npx clawhub install obsidian mcporter clawhub

# FE 角色
npx clawhub install obsidian

# BE 角色
npx clawhub install obsidian tmux

# QA 角色
npx clawhub install obsidian nano-pdf

# OPS 角色
npx clawhub install mcporter tmux obsidian
```

## 💡 使用建议

1. **优先级排序**: 先安装 🔴 高优先级技能

3. **配置优先**: 安装后立即配置 API Key 和认证
4. **团队同步**: 安装后通知团队成员
